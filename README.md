# 🧪 Reusable Unity Assets 

A collection of pre-assembled, reusable prefabs and assets ready to drop directly into your Unity projects.

## ⚠️ Important Compatibility Note
> **All assets in this library were created using the Universal Render Pipeline (URP) template.**
> * **Make sure to use the URP template** in your target project before importing these assets.
> * If you import them into a Standard / Built-In Unity template, the materials will break and appear **bright pink**.

---

## 🚀 How to Download & Install

1. Go to the **Releases** section on the right side of this repository page.
2. Download the **`.unitypackage`** file for the asset bundle you want to use.
3. Open your target **Unity URP Project**.
4. Drag and drop the downloaded file directly into your Unity **Project window**.
5. Click **Import** in the pop-up window that appears.
6. Open the newly imported folder, find the prefab, and drag it straight into your scene!

---

## 📦 What's Included in the Asset Packages
* **Pre-assembled Prefabs** (Ready to use with meshes and materials already configured)
* **3D Mesh Files** (.obj / .fbx)
* **URP Materials & Textures**
* **Associated Scripts** (Where applicable)
