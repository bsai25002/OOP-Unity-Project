## Project Overview

It pours water from one beaker to another. The water level is correctly kept at 20 cm cubed. 
It has watch glass and salt components. 
The salt isn't yet correctly grabbable, and keeps bouncing off the watch glass. The source beaker doesn't empty when it pours the liquid. 
The liquid is a cylinder itself, with a clear-water-like material added to it. 
For the receiving beaker, we put an empty collider object above its rim to detect when the other beaker's pourPoint touches it. 
For both beakers, we have pour point objects set at the beaks. 
The setup involves a probuilder plane made into a room and roof, and one for table. 

The weighing balance, calculator, reading notebook - couldn't be added.
